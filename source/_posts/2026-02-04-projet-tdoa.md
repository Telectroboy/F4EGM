### Message par Anonyme, 2026-02-04

1) AEDA — Plateforme crowd-sourcée RTL-SDR pour analyse spectrale & TDoA C’est une plateforme qui permet d’utiliser plusieurs dongles RTL-SDR répartis (typiquement sur des Raspberry Pi) pour faire des analyses spectrales et des localisations TDoA à partir d’une interface Web.
Elle supporte : analyse spectrale, détection de modulation (WBFM, NBFM, DMR, P25), localisation par TDoA en utilisant au moins trois nœuds bien géométrisés autour de la source. Documentation AEDA (fonctionnement et configuration) :
[https://logarithms-corp.com/documentation](https://logarithms-corp.com/documentation) Code source de la node AEDA (en beta sur GitHub) :
[https://github.com/Logarithms-Corp/aeda-node](https://github.com/Logarithms-Corp/aeda-node) 2) SDR-TDOA-DF — Direction Finding / TDoA avec plusieurs RTL-SDR Un projet open source sur GitHub qui propose des scripts Python pour mettre en place une localisation TDoA basique avec plusieurs RTL-SDR connectés à un PC (ou RPi).
Bon point de départ pour expérimenter la TDoA dans un cadre radioamateur. Lien du dépôt :
[https://github.com/ArgusSDR/SDR-TDOA-DF](https://github.com/ArgusSDR/SDR-TDOA-DF) 3) tdoa-geolocation — Géolocalisation TDoA distribuée Projet structuré pour faire de la multilateration TDoA avec plusieurs récepteurs RTL-SDR. Il est conçu pour être déployé sur plusieurs RPi et permet une architecture distribuée.
Idéal pour un réseau de stations radioamateurs réparties.
(Ce projet n’est pas un service mais une base logicielle que vous pouvez adapter.) Lien du dépôt :
[https://github.com/KX0U-Jim/tdoa-geolocation](https://github.com/KX0U-Jim/tdoa-geolocation) 4) Système de localisation TDOA de référence (DC9ST / 2017) Même si c’est plus ancien, c’est souvent cité comme base d’implémentation et il existe des scripts Matlab/Octave associés.
Peut servir de référence technique pour comprendre et comparer avec AEDA ou SDR-TDOA-DF. Page d’origine :
[https://panoradio-sdr.de/tdoa-transmitter-localization-with-rtl-sdrs](https://panoradio-sdr.de/tdoa-transmitter-localization-with-rtl-sdrs)** J'aime** Je n'aime pas
